Title: Scheduling Application
Purpose: To create customers and to schedule customer appointments.
Author: Julian Johnson
Contact Info: jjo2644@wgu.edu
Version: v1.0
Date: 18-May-2022
IDE: IntelliJ IDEA 2021.3.3 (Community Edition)
JDK: Java SE 17.0.2
JavaFX Version: JavaFX-SDK-17.0.2

Directions for use:
1. Download the appropriate JavaFX libraries, load the project to IntelliJ IDEA, and run the program.
2. The first screen will be the login form. Use your user ID (not username) and password when logging in.
3. After successfully logging in, schedules, appointments, and reports tabs will be made available to use.

MySQL Driver Version: mysql-connector-java-8.0.27
